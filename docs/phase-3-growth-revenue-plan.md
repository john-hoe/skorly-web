## 0. 为什么现在重排规划：一个不可错过的时间事实

**2026 世界杯 6 月 11 日开赛（明天）→ 7 月 19 日决赛，共 39 天。**

这是本站从建站第一天起瞄准的唯一流量洪峰。印尼/越南/菲律宾的足球搜索量在世界杯期间通常放大 5–20 倍，且 `hasil`（赛果）、`skor langsung`（实时比分）、`jadwal hari ini`（今日赛程）类词的流量**集中在比赛进行中和赛后 2 小时**。

总原则：

1. **赛中（6.11–7.19）一切为流量让路**。
2. **变现通道必须立刻打开**：广告审核需要数天到数周——这是目前**唯一还完全没动**的主线。
3. **赛后悬崖必须提前布防**：站点目前 100% 世界杯导向，7 月 20 日流量会断崖。

三个阶段：


| 阶段          | 时间          | 主题                    |
| ----------- | ----------- | --------------------- |
| **S1 赛中冲刺** | 6.11 – 7.19 | 内容新鲜度提频 + 开广告 + 建分发渠道 |
| **S2 赛后承接** | 7.20 – 8 月底 | 转会窗 + 新赛季内容，接住留存用户    |
| **S3 常态运营** | 9 月起        | 多联赛常青站 + 收入结构优化       |

## 2. 流量增长（Traffic）

### 2.1 SEO：吃透"比赛日意图"（P0）

赛前（预测/阵容）✅、赛中（实时比分）✅ 基本就位，**剩下的全部火力对准"赛后新鲜度"**：

- **比赛日部署提频（新的第一优先工程项）**：把"出稿→上线"的延迟从 ≤24h 压到 ≤30min。方案（按 spec 化后交执行方选型）：
  - news.yml 增加比赛日 schedule（如世界杯期间每 30–60 分钟一班，仅当 DB 有新 published 文章才执行构建+部署，否则秒退）；或由 jobs Worker 在 recap 发布后触发 `workflow_dispatch`。
  - 维持既有节流原则（部署 ≥30 分钟一次），避开构建竞态（`concurrency` 已有）。
  - 完成判据：终场哨后 ≤30min，4 语言 recap 出现在线上且进 news-sitemap。
- **真实比赛日验证（6.11 首日）**：盯 `/api/live` 数据流、KV 快照节奏、`apiq` 配额计数、降级阈值；用后台 `/admin/matches` 做单场补抓演练。完成判据：首个比赛日全程比分实时、配额 <2000/日。
- `/jadwal`、`/nonton` 已上线 → 转为**运营项**：确认进 sitemap/导航入口/内链（赛程页→猜比分、文章→nonton），无需新开发。
- Discover：metadata 已修复 → 转为**观察项**，GSC 里盯 Discover 报表，有量后再加码大图变体。

### 2.2 留存回访：节奏化运营（P1，功能已有，接排期即可）

- 早 08:00 当地时间：今日赛程 + 昨日竞猜结果 + 排行榜变动 → Push + Email（接现有 cron/后台管道，做成自动排期而非手动按钮）。
- 赛前 1 小时：单场预告 + 猜比分入口 → Push。
- 完成判据：赛中每日自动发出；PostHog 可见 push/email 渠道回访占比（埋点已在）。

### 2.3 站外分发：社交短内容流水线（P1，不变）

- 渠道：TikTok（id 主账号）+ Instagram + X + **WhatsApp Channel**。
- 内容全部复用现有资产（海报/比分卡/4 语言文案/bracket 分享图），人工只做发布排期。
- 先 id 单语验证，UTM 量化（埋点已上线，渠道归因现成）；7 日内社交新访客 >10% 再扩 vi。
- 完成判据：账号建立、赛中每日发布、GA4/PostHog 可区分社交渠道流量。

### 2.4 度量底座 ✅ 已上线（原 Spec A）

转为**使用阶段**：在 PostHog 建好激活漏斗（pageview → predict/bracket → signup → push/email）与 WAP 看板，S1 每周复盘一次，作为 §5 所有取舍的依据。

## 3. 商业化（Revenue）

> 合规底线不变：**不显示赔率、不接博彩导流、不碰盗播**。

### 3.1 展示广告（P0，目前唯一完全未动的主线，今天就启动）

- **立即申请 Google AdSense**。准备项（现在比 v1 时更顺——consent 门控、隐私页、security headers 都已上线）：
  - `ads.txt` 上线（当前 404）。
  - Consent Mode v2 与现有 consent banner 对接（banner 已生产可用）。
  - AdSense 后台屏蔽"博彩"类目（印尼合规关键）。
  - 广告位规划（克制，保 CWV/Discover）：文章页 in-article 1–2 + 文末 1；比分/赛程页移动端 anchor 1。首屏不放大广告。
- 现实预期：印尼/越南 RPM ~$0.3–1.5；S1 意义是**开通道、攒数据**。
- 升级路径（S3）：≥5 万 PV/月后评估 Ezoic/Monumetric；长期 Raptive 级（en-PH 英语流量的长期价值所在）。
- 完成判据：过审出广告；ads.txt 200；拒绝 consent 时不加载广告；CWV 不回退。

### 3.2 联盟/导购（P1，载体页已存在，接链接即可）

- **观赛联盟**：`/nonton` 页已上线 → 通过 Involve Asia / AccessTrade 接各市场官方转播平台 CPA 链接，替换/包装现有官方导流。意图最强、单价最高的位置。
- **球衣/周边电商联盟**：Shopee/Lazada/Tokopedia 联盟，挂球队页 + "XX 队球衣"导购文章（新闻管线低成本产出，48 队 × 多语言）。
- 披露合规：`rel="sponsored"` + 页面披露声明。
- 完成判据：≥1 联盟网络过审；nonton 页与导购页带追踪链接上线；后台可见点击/转化。

### 3.3 直客赞助 / Media Kit（P1，零工程量）

- `/[locale]/iklan` Media Kit 页（当前 404）：定位、4 语言覆盖、赛中流量截图、广告位、`business@skorly.cc`。
- 世界杯期间主动 outreach：本地体育 App、球衣电商、流媒体平台市场部（筛掉博彩壳）。单条赞助贴 $50–500。
- 完成判据：页面上线；发出 ≥10 封 outreach；记录回复率。

### 3.4 付费会员（P2，S3 再做，赛中不做）

- S1/S2 premium 保持免费换注册。S3 用攒下的漏斗数据（现已在采集）决策付费化：AI 深度方案 + 无广告 + mini-league 工具包，定价 ≈$1–2/月；支付选 merchant-of-record（Paddle/Lemon Squeezy）或本地 Midtrans。
- 红线：付费内容仍是"编辑分析"，不滑向"卖料"。

### 3.5 Newsletter 赞助（P2）

订阅列表 >3000 后启动，邮件头部赞助位 $20–100/发。
